🛒 Django E-commerce Web Application

📌 Description

This is a simple e-commerce web application built using Django. Users can browse products, add items to cart, and proceed to checkout.

🚀 Features

- View all products
- Add to cart
- Remove from cart
- Checkout page
- Product images and pricing

🛠️ Tech Stack

- Python
- Django
- HTML, CSS
- SQLite

📂 Project Structure

- store/ → app logic
- templates/ → HTML files
- media/ → product images

▶️ How to Run

1. Clone the repository
2. Run: python manage.py runserver
3. Open: http://127.0.0.1:8000/

📌 Future Improvements

- Search functionality
- User login/signup
